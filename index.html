Kho APT cho FlyEnv

Đây là kho APT (dành cho Ubuntu/Debian) để cài đặt và cập nhật FlyEnv.

1. Thêm kho APT

Mở terminal và chạy:

echo "deb [arch=amd64 trusted=yes] https://flyenv.tannhatcms.io.vn ./" | sudo tee /etc/apt/sources.list.d/flyenv.list
sudo apt update

2. Cài đặt FlyEnv

sudo apt install flyenv

3. Nâng cấp

Mỗi khi có phiên bản mới, chỉ cần chạy:

sudo apt update
sudo apt upgrade

Lưu ý

  • Kho này hiện chưa ký GPG, nên sử dụng trusted=yes.
  • Các gói .deb thực tế được lưu trên GitHub Releases, apt sẽ tự động tải về.
